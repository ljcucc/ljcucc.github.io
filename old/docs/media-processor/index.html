Introduction

Meida Processor is a minimalist photo editor.
Which using less computing resources and clear interface to perform operation. Which is programmable and batchable and easy to learn.

That inspired by Ronin*

Get Started

Go to live demo* to try it out.

Meida Processor is designed to user friendly which also base on layers, so the concepts are very similar to photoshop.

Interfaces

[Helper] panel at the bottom of the left, is a reference and docs. try to click any line, [Helper] will showing the reference of current line you editing.

Here an example, You can try to type this in your code editor by press [Ctrl]+[R]: 

- background: black
+ layer1: blank
  - shape: rect
  - fill: white
Then you'll see a white block in a black background.

Layers & Properties

In MeidaProcessor,which have "layers" that containing image or contents. But MediaProcess using plain text to perform it. Here an layer which position at the top, left. 

+ layer1: blank
  - x: 0
  - y: 0
the line started with "+" is the layer, "layer1" is the name. "- x", "- y" is the properties which under the "layer1".

Comment in codes will look like this: 

- foo: bar ; this is comment
